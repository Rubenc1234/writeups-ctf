# TryHackMe Writeups

This repository contains my personal writeups and solutions for TryHackMe (THM) rooms and challenges. Each entry documents the methodology, tools, and technical steps taken to complete the objectives.

## Repository Structure

* **TryHackMe/**: Detailed writeups categorized by room type and difficulty.
    * **AdventOfCyber/**: Yearly participation in the Advent of Cyber events.
    * **Challenges/**: Machine-based challenges categorized by difficulty (Easy, Medium, Hard).

## Methodology

Each writeup typically covers:
1. **Reconnaissance**: Nmap scans, service enumeration, and directory brute-forcing.
2. **Exploitation**: Vulnerability analysis and gaining initial access.
3. **Privilege Escalation**: Post-exploitation and gaining root/administrator access.
